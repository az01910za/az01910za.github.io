Linux安装及服务

wget下载

1.通过源码安装 ./configure make make install

./configure:解压完成正式安装，跳转到解压目录，这个命令运行当前脚本

make:上一步生成makefile文件，make生成目标文件。

make install将可执行程序安装到正确位置

make clean根据makefile清理临时文件

make uninstall根据makefile删除安装后程序，需root用户

2.通过rpm安装 rpm -ivh xx.rpm 可以安装gcc, mongodb

-i安装指定的.rpm文件
-v查看更细部的安装信息画面
-h以安装信息列显示安装进度
-U用指定的.rpm文件升级同名包

3.通过yum安装 yum serch yum install 自动处理软件相依属性问题

yum remove nmap不删除依赖
yum groupinstall “Development Tools” -y

安装nginx

mkdir nginx
cd nginx/
wget http://192.168.100.195:8000/nginx-1.17.2.tar.gz
tar zxvf nginx-1.17.2.tar.gz
cd nginx-1.17.2
more README
./configure
// –conf-path=/etc/nginx/conf/nginx.conf –sbin-path=/usr/bin/nginx出错百度解决
yum install pcre-devel pcre
yum install zlib zlib-devel
make
make install
/usr/local/nginx/sbin/nginx
systemctl stop firewalld
http://ip
scp -r ./* root@ip:/usr/local/nginx/html/
httpd

服务

service network restart

service -> systemctl (centos7)
systemctl restart 服务名称
systemctl stop 服务名称
systemctl start 服务名称
systemctl enable 服务
systemctl disable sshd
