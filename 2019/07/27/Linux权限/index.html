chown

chown root:root(用户：组) 1（文件夹）
chown 将指定文件的拥有者改为指定的用户或组，用户可以是用户名或者用户ID；组可以是组名或者组ID；文件是以空格分开的要改变权限的文件列表，支持通配符。
将文件 file1.txt 的拥有者设为 runoob，群体的使用者 runoobgroup :
chown runoob:runoobgroup file1.txt
将目前目录下的所有文件与子目录的拥有者皆设为 runoob，群体的使用者 runoobgroup:
chown -R runoob:runoobgroup

chmod

u用户 g组 o其他用户 a所有用户 +赋予 -取消 =
chmod ug+w 1 chmod 777 1
chmod 777 /var/home/userid/cc 修改文件夹
chmod -R 777 /var/home/userid/cc 修改文件夹及子文件夹

chgrp

该命令改变指定指定文件所属的用户组。
– R 递归式地改变指定目录及其下的所有子目录和文件的属组。

出现“不在sudoers文件中。此事将被报告。”问题解决

1.以root用户登录系统，进入/etc目录，查看sudoers文件的权限
cd /etc ll sudoers
2.改sudoers文件的权限使其不是只读
chmod 777 sudoers
3.编辑sudoers文件，添加abc用户为所有可执行权限
root ALL=(ALL) ALL
abc ALL=(ALL) ALL
如果在使用sudo不输入密码在编辑sudoers文件的时候添加NOPASSWD: 即可
root ALL=(ALL) ALL
abc ALL=(ALL) NOPASSWD: ALL

  1. 然后修改回sudoers文件权限为440
    chmod 440 sudoers
