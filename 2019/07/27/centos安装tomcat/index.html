centos安装tomcat

写于 2019 年 07 月 27 日 23 时 13 分
更新于 2019 年 07 月 27 日 23 时 30 分
大纲
  1. 1. Powershell登陆
  2. 2. centos7开启ssh服务

[root@localhost 下载]# tar -zxvf tomcat.tar.gz
[root@localhost 下载]# cp -R apache-tomcat-8.5.43/ /opt
[root@localhost 下载]# cd /opt/usr/local
[root@localhost local]# mkdir java
[root@localhost 下载]# cp -R apache-tomcat-8.5.43 /usr/local/java
[root@localhost local]# cd /java/apache-tomcat-8.5.43/conf
[root@localhost conf]# vi server.xml 修改端口

[root@localhost apache-tomcat-8.5.43]# cd bin/
[root@localhost bin]# ./startup.sh
[root@localhost bin]# systemctl start firewalld
[root@localhost bin]# firewall-cmd –zone=public –add-port=80/tcp –permanent
[root@localhost bin]# firewall-cmd –reload
centos网页上127.0.0.1 进入Host Manager
在/opt/tomcat中
[root@localhost apache-tomcat-8.5.43]# cd conf/
[root@localhost conf]# vim tomcat-users.xml

[root@localhost apache-tomcat-8.5.43]#cd bin
[root@localhost bin]# sudo ./shutdown.sh 30秒左右
[root@localhost bin]# sudo ./startup.sh
centos网页上127.0.0.1
Host Manager登陆
在外面ip:8080出来Apache界面

进入ManagerApp
[root@localhost apache-tomcat-8.5.43]# cd /webapps/manager/META-INF
[root@localhost META-INF]# cp context.xml context.xml.bak723
[root@localhost META-INF]# vim context.xml

[root@localhost apache-tomcat-8.5.43]# cd bin/ 下次登陆
[root@localhost bin]# ./shutdown.sh
[root@localhost bin]# ./startup.sh

Powershell登陆

PS C:\Windows\system32> ssh -l root ip

centos7开启ssh服务

开启ssh服务需要root权限，先用root账户登陆
先检查有没有安装ssh服务：rpm -qa | grep ssh
如果没有安装ssh服务就安装: yum install openssh-server
安装好后在ssh配置文件里进行配置: vim /etc/ssh/sshd_config
将Port 22前#去掉，PermitRootLogin yes
修改完后家目录用 /bin/systemctl start sshd.service 开启ssh服务,这个命令没有回显
开启后用 ps -e | grep sshd 检查一下ssh服务是否开启
再用netstat -an | grep 22检查一下22端口是否开启
将ssh服务添加到自启动列表中：systemctl enable sshd.service
Xshell就可以连接了
